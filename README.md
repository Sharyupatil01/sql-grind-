# SQL-Grind 


A structured SQL interview preparation repository covering
SQL fundamentals to advanced topics.

## Topics Covered
- SQL Fundamentals
- Aggregations & HAVING
- Joins (Single & Multi-Table)
- Subqueries & Correlated Subqueries
- Window Functions

## Each folder contains:
- Concept-wise problems
- Clean SQL solutions
- Thought-process based explanations
